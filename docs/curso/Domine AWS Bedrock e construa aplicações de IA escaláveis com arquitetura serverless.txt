Sobre o Curso

Domine a criação de aplicações de Inteligência Artificial Generativa utilizando a arquitetura serverless da AWS. Este curso oferece uma abordagem prática e completa, focando no AWS Bedrock e sua integração com outros serviços da AWS para construir soluções escaláveis e eficientes.

O que você vai aprender

Fundamentos Teóricos

Conceitos essenciais de IA Generativa

Arquitetura serverless na AWS

AWS Bedrock: recursos e capacidades

Integração entre serviços AWS

Melhores práticas de desenvolvimento

Projeto Prático

Identificação do problema: Definição de um caso de uso real

Arquitetura da solução: Design de sistema serverless

Implementação: Desenvolvimento hands-on

Integração: Conectando Bedrock com outros serviços AWS

Deploy e monitoramento: Colocando em produção

O que você aprenderá
Configurar e gerenciar contas AWS com boas práticas de segurança e governança usando IAM Identity Center
Implementar soluções de IA Generativa usando AWS Bedrock, incluindo modelos de fundação e Knowledge Bases
Criar workflows inteligentes com Bedrock Flow e desenvolver agentes conversacionais com Bedrock Agent
Construir aplicações completas integrando WhatsApp, API Gateway, bancos de dados e monitoramento
Há algum requisito ou pré-requisito para o curso?
Conhecimento básico de programação (Python e Angular)
Familiaridade com conceitos de cloud computing (desejável, mas não obrigatório)
Computador com acesso à internet para usar os serviços da AWS
Disposição para aprender sobre IA Generativa e serviços AWS
Para quem é este curso:
Este curso é ideal para desenvolvedores, arquitetos de soluções e profissionais de TI que desejam implementar soluções de IA Generativa na AWS. É perfeito para quem quer aprender desde a configuração básica de contas AWS até o desenvolvimento de aplicações completas com Bedrock. Também é adequado para empreendedores e product managers que querem entender as possibilidades técnicas da IA Generativa para criar novos produtos e serviços.
